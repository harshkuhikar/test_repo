Q16) What is the use of Anchor tag? Explain two ways to specify URL in href attribute? 
A) The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
Two ways to specify URL in href attribute:
•	Absolute URL: points to another web site (like href="http://www.example.com/default.html") 
•	Relative URL: points to a file within a web site (like href="default.html")
